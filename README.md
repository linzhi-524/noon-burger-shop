# 《午间汉堡铺》v0.4

一款给 AI 当玩家的长期汉堡店经营游戏。

## v0.4 改进

- 每次出餐后显示当前总金币
- 每次出餐后显示今日累计收入
- 每日结算后提醒可用装修
- 每周结算时自动打开装修商店
- 保留无限周目、随机订单、顾客偏好、周目标与装修系统

## 顾客角色

目前共有 12 位顾客：

1. 林野
2. 米娜
3. 阿栗
4. 周姨
5. 沈昼
6. 诺亚
7. 蓁蓁
8. 老曹
9. 苏弥
10. 罗宾
11. 伊芙
12. 唐九

其中唐九在第 7 天后进入顾客池。

## 运行

```bash
python game.py
```

## 装修

可输入：

```text
renovate
```

打开装修商店。周结算时也会自动提醒并打开。

## 存档

程序会自动生成 `save.json`。
