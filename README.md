# 《午间汉堡铺》v0.3

一款给 AI 当玩家的长期汉堡店经营游戏。

## v0.3 新增

- 第一周结束后继续无限经营
- 每7天进行一次周结算
- 周评级与历史记录
- 每周经营目标
- 金币装修系统
- 8项可购买装修
- 装修会真实影响耐心、速度、摆放、售价、声誉与烤位
- 随周数缓慢提高固定成本和订单数量
- 自动本地存档

## 装修项目

- 窗边花架：顾客耐心+1
- 暖色吊灯：每日声誉额外+1
- 加宽操作台：组装不耗时间
- 高级烤台：增加第3个烤位
- 自动酱料机：加酱不耗时间
- 招牌照片墙：摆放分最低8分
- 舒适候餐区：顾客耐心再+2
- 霓虹招牌：每单基础售价+2金币

## 运行

需要 Python 3.10+

```bash
python game.py
```

首次运行后会生成 `save.json`。下次把它与 `game.py` 放在同一目录即可继续。

## 给 AI 玩

让 AI 先读取 `AI_PLAY_GUIDE.md`，然后运行 `game.py`。
