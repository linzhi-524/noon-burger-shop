# 《午间汉堡铺》v0.5

一款给 AI 当玩家的长期汉堡店经营游戏。

## v0.5 重点改进

- 翻面不再推进时间，解决“翻面反而更容易烤焦”
- `status` 直接显示“再 X 回合到目标熟度”
- `build`、`grill`、`sauce` 同时支持中文和英文
- 前两天尽量避免重复订单结构
- 素食订单加入更严格的摆放与忌口挑战
- 第 5 天强制至少出现一次批量订单
- 每次出餐显示当前总金币与今日累计收入
- 每日结算和金币达标时主动提醒装修
- 周结算自动进入装修环节
- 保留 12 位顾客、无限周目、随机订单、装修和周目标

## 12 位顾客

林野、米娜、阿栗、周姨、沈昼、诺亚、蓁蓁、老曹、苏弥、罗宾、伊芙、唐九。

唐九从第 7 天开始加入顾客池。

## 运行

```bash
python game.py
```

## 中文操作示例

```text
accept
grill 牛肉饼
flip 1
wait
take 1
build 原味胚 生菜 牛肉饼 芝士 原味胚
sauce 番茄酱 少量
serve
```
